jQuery.BlockUI是一个jQuery插件，页面或元素覆盖层

集成

在html文档中的<head>部分，引入jquery.blockui.min.js文件。 

<script src="jquery.min.js" ></script>
<script src="jquery.blockui.min.js" ></script>

用法 

<script>		
	// invoke blockUI as needed 
	$(document).on('click', '#myButton', function() {
		$.blockUI();
	});
</script>
