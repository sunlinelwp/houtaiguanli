jQuery.JSON是一个jQuery插件，提供简单的JSON和Javascript对象间转换

集成

在html文档中的<head>部分，引入jquery.json.min.js文件。 

<script src="jquery.min.js" ></script>
<script src="jquery.json.min.js" ></script>

用法

将Javascript Object转换成JSON对象 

$.toJSON(myObject);
$.toJSON({ foo: 'bar' });

将JSON对象转换成Javascript Object 

var obj = $.evalJSON('{ "foo": "bar" }');
